# NYC Schools (20230513-UdaybhaskarVS-NYCSchools)

This is a sample Android native app build
with
- DarkTheme
- LightTheme

using
- kotlin
- ViewModel
- LiveData
- Coroutines
- Jetpack Navigation
- Room
- Dagger2
- Retrofit2
- RXJava2
- Mockito.

This app has mainly two screens
- List of Schools
- Additional School Information with SAT Results
